## 🛠️ 核心技术栈

本项目使用纯 HTML + CSS 构建，无依赖、加载快、SEO 友好：

- **HTML5** 语义化标签
- **Schema.org JSON-LD** 结构化数据（GEO + SEO 双重优化）
- **Dublin Core** 元数据（学术/AI 引擎友好）
- **Open Graph + Twitter Card** 社交分享优化
- **GEO 协议** `ai-content: allowed` 允许 AI 搜索引擎抓取
- **响应式设计** 移动端友好
