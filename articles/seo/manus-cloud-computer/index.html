Meta收购的那个Manus，今天推出了一款让普通人也能跑AI智能体的产品

2026年5月3日 · 约1700字 · 读完需要5分钟

事情是这样的。

前两天刷ProductHunt，看到一个叫"Cloud Computer by Manus"的产品。Manus？就是那个被Meta收购的AI Agent公司。他们今天在ProductHunt上众筹了一台"永远在线的云端机器"，专门跑 Bots、软件、脚本，24小时不间断。

我看完就愣住了。

为什么愣住了？因为这个问题我踩过坑，而且踩得很疼。

我自己踩过的坑

去年我想用AI做一个自动化的内容聚合机器人，逻辑很简单——每天定时抓取几个AI领域的新闻源，用LLM总结之后发到飞书上。本地测试，两周就跑通了。

然后就没有然后了。

我的电脑不可能24小时开着。那段时间我每天早上起来第一件事就是打开电脑启动脚本，晚上睡觉前再关掉。坚持了大概一个月，人都快崩溃了，最后直接放弃。

这不是我一个人遇到的问题。只要你想用AI做"自动化"的事情，一定会碰到这堵墙——你的电脑不能永远在线，但AI任务偏偏需要7×24小时不间断运行。

传统解法是租云服务器，自己配置环境、写启动脚本、处理依赖问题。有技术背景的人觉得这不算什么，但我这种"普通AI用户"，光是搞懂怎么SSH登录就已经耗尽了所有热情。

Cloud Computer就在这里插了一脚。

它解决的是什么？

根据ProductHunt上的介绍，Cloud Computer就是一台永远在线的云端电脑。你有了一台24小时运行的专用机器，可以往上面部署 Bots、脚本、应用、数据库。

重点在于"永远在线"这四个字。

你平时用的ChatGPT、Claude，都是"按需调用"。你发一个请求，AI处理完返回给你，这次交互就结束了。下次再发一个请求，重新开始。

但很多场景需要AI"一直在跑"。自动客服机器人需要随时响应用户咨询，数据监控系统需要定时抓取并告警，内容聚合机器人需要每天定时整理信息。这些场景不是"一问一答"，而是"持续运行"。Cloud Computer就是为这种场景设计的。

跟租服务器有什么区别？

你可能会说，这不就是云服务器吗？AWS、阿里云都能租。

对，但不一样。

租过云服务器的人知道，整个流程是这样的：选配置、装系统、装运行环境、配网络、写启动脚本、调试。每一步都可能踩坑，每一步都需要技术知识。

Cloud Computer的宣传语是"No DevOps, no server setup, no coding required"。不需要懂任何这些，用大白话描述你要做什么，它帮你搞定。

我不是说他真的像宣传的那么简单。"no coding"大概率指不需要写传统代码，配置AI Bots的过程肯定还是需要学习的。但相比自己租服务器搭环境，门槛确实低了一个数量级。

一个具体的对比：

你自己租云服务器跑一个Telegram Bot，完整流程大约需要：云服务器月费（50元起）+ 环境配置（2到4小时）+ 调试（可能需要几天）+ 持续维护。

用Cloud Computer：描述你要做的事情，它帮你把Bot跑起来，24小时在线。不需要懂Linux，不需要会写启动脚本，不需要处理依赖冲突。

对普通人做副业意味着什么

这是我觉得最值得关注的部分。

我一直觉得AI副业最大的门槛不是"没有好的AI工具"，而是"有了工具之后不知道怎么让它持续运行"。很多人有想法，"我想做一个自动xxx的机器人"，但一到"怎么让它7×24小时跑"，就卡住了，然后就没有然后了。

Cloud Computer解决的就是这个问题。

我能想到的方向有这么几个。自动化客服外包，帮小商家搭建AI客服系统，把 Bots部署在Cloud Computer上，7×24小时在线自动回复。Multi-Agent多智能体的方向，用多个AI Bots协同处理不同任务，实现真正的自动化工作流。内容自动化运营，不管是公众号还是小红书，很多内容运营工作本质上是重复的，找素材、整理、格式化、发布，用AI Bots自动化这个流程，你只负责审核和调整。还有信息聚合服务，做一个垂直领域的信息订阅，比如每天早上9点推送今日AI工具新品，用Cloud Computer跑一个聚合Bot，定时抓取定时推送，完全自动化。

我自己没有把这几个方向全部测试过，这些都是看产品介绍和用户评论之后推导的。但核心逻辑很简单——Cloud Computer解决的是"AI持续运行"的问题，这个问题以前只有懂技术的人才能解决。

Meta在下一盘什么棋

说到Manus，必须提一下背景。Manus在2025年被Meta收购了。

Meta不是那种"随便投资一下试试"的公司风格，他们收购一家AI Agent公司，一定是看中了什么。Manus的定位是"通用AI Agent"，不只是帮你回答问题，而是真的帮你完成任务。做研究、写代码、搭建网站、处理数据、安排行程，几乎涵盖了知识工作的方方面面。

加上Cloud Computer这条产品线，Meta现在有两张牌：一个是面向企业的AI Agent平台（Manus Team），一个是把AI能力普惠给普通人的基础设施（Cloud Computer）。

当普通人也能用自然语言部署一个7×24小时在线的AI Bots的时候，会发生什么？

这个图景比"ChatGPT能回答问题"有意思得多。

我的判断

Cloud Computer目前还处于早期阶段。ProductHunt上只有几百个投票，没有太多真实用户反馈。定价信息也没有完全公开，Manus订阅计划从每月20美元起，但Cloud Computer作为独立产品的价格还不清楚。

如果你对AI副业感兴趣，这个方向值得关注，但不要急着冲。先观察一段时间，等第一批真实用户反馈出来之后再决定要不要下场。

如果你已经有明确的AI副业方向，在做内容自动化或者客服外包这类，Cloud Computer值得你花时间研究一下。它的定位很清晰，解决"AI持续运行"的问题，而这恰恰是很多副业项目最难解决的环节。

说实话，这类产品的出现让我对"AI副业"这个概念更有信心了。以前很多人说AI副业就是割韭菜，一个重要原因是大部分AI工具都需要持续人工干预，很难真正规模化。Cloud Computer这类基础设施的完善，让"一个人加AI Bots"这种轻资产副业模式变得更现实了一点。

磨平一些信息差，这是我一直相信的事情。
