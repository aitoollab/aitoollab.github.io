Google Gemini 3.6 发布，API 价格打骨折，普通人怎么用最便宜的模型赚到钱

昨天 Google 一口气发了三个新模型，Gemini 3.6 Flash、3.5 Flash-Lite、还有 3.5 Flash Cyber。我大概看了一下，让我兴奋的不是什么技术突破或者 benchmark 又涨了多少，而是那个价格。

3.6 Flash 输入 $1.5/百万 token，输出 $7.5/百万，比 3.5 Flash 还便宜。3.5 Flash-Lite 直接 $0.3/百万输入，$2.5/百万输出，你想想看，现在模型市场这是什么水平。

说真的，这个价格已经低到你可以拿它来跑过去根本不敢想的事情。

很多人觉得模型降价跟自己没关系，但我跟你讲，这想法正好错过了最大的机会。

想明白这事就得理解一个逻辑，AI 副业最大的成本从来不是人工，是 API 调用费。你做一个聊天机器人挂到淘宝上接单，每个月跑几千次对话，API 账单占你总成本的 60-70%。Google 把价格打到这个程度，等于直接给你提了利润空间。

顺着这个思路，我梳理了三个普通人现在就能上车赚钱的方向。

方向一，用 Flash-Lite 跑高并发的对话服务。

$0.3/百万输入的价格，你跑一千次对话的成本只要几毛钱。你完全可以做一个 24 小时在线的小助手丢到闲鱼或者淘宝上，帮人答疑解惑、陪聊、或者做简单的咨询。我自己前两天试了一下，用 3.5 Flash-Lite 搭了一个装修咨询机器人，丢到一个业主群里试跑，成本几乎可以忽略。

方向二，用 3.6 Flash 做文档处理和分析。

这次 3.6 Flash 在解析文档、分析图表、撰写报告方面的进步很大。好多律所和金融公司已经在用了。你可以把这个包装成服务，帮小企业处理合同、整理报表、出分析报告。

一块钱一页的合同审查服务，跑遍中小城市的律所，一天接三五单不愁。成本呢？一份合同几毛钱的 API 费。

方向三，最炸的，用低模型成本套个人品牌。

这个可能你没想到。价格降到这个程度，用 3.6 Flash 批量生成某个垂直领域的信息产品，每日行业简报、专业问答合集，这些东西本身就能卖钱。

不是哥们，这跟那些批量洗稿的生成完全不一样。你要做的是在你自己确实懂的那个小领域里，用模型帮你提效，产出有个人判断的东西。成本低到几乎为零，但用户为你的判断力付费。

当然我也得说实话，不是模型便宜就自动赚到钱。我自己踩过最深的坑，上来就想搞个大平台，花了两周搭架子一个客户没见着。后来从最简单的场景开始，闲鱼挂个「AI 写小红书文案」的服务，当天就有人问。

你要做的事其实很简单，先想清楚你的客户在哪，再想模型能帮你解决什么问题。反着来必死。

说到这，大模型价格一路在降，这不是新趋势。但 Google 这次把价格打到这个程度，加上 350 tokens/秒的处理速度，以前那些「成本太高做不了」的服务方向，都值得重新算一遍账。

你可以回想一下，每一次变革的机会窗口都差不多，技术成本暴跌，有人先跑通，后来者看懂再冲，然后下一轮又来。现在这波刚刚开始。

大时代啊，朋友们。早点动手。

常见问题

Gemini 3.6 Flash 比之前便宜多少？

3.6 Flash 的 API 定价为输入 $1.5/百万 token，输出 $7.5/百万 token，比 3.5 Flash 还低。而且它在编码和知识工作上的性能更好，输出 token 用量还减少了 17%。实际跑一个任务的总成本比 3.5 Flash 低了 30-40%。非常适合用来搭建成本敏感的 AI 服务。

普通人怎么用 Gemini 3.5 Flash-Lite 做 AI 副业？

3.5 Flash-Lite 定价 $0.3/百万输入 tokens，速度达 350 tokens/秒，适合做客服机器人、内容批量生成、数据整理等高频场景。你可以在闲鱼或淘宝上挂 AI 服务接单，成本极低。比如做一个装修咨询机器人丢到业主群里，一个月 API 费用也就十几块钱。

AI API 降价对副业有什么实际影响？

API 降价直接降低了做 AI 服务的边际成本。过去一个 AI 小助手的月度 API 账单可能几百上千，现在用 Gemini 3.5 Flash-Lite 可以压到几十块。以前那些因为成本太高不做的服务方向（比如 24 小时在线客服、个人知识库问答）现在都能跑通了。利润空间大了，自然好变现。
