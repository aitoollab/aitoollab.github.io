菲尔兹奖得主27年没搞定的App AI几个小时做完了 帮人App考古是个冷门副业

发布于 2026-07-13 · AI热点追踪

这两天HN上有个帖子特别火，422分。Terry Tao，就是那个菲尔兹奖得主、全世界最聪明的数学家之一，发了一篇博客。

讲他最近用AI Coding Agent干了一件事。1999年他写了一批Java Applets，给教学用的，可视化复分析、线性代数什么的。后来浏览器不认Java Applet了，这些代码就烂在了硬盘里。

前阵子他试了试用AI，让Agent把这些Applets从Java迁移到JavaScript。几个小时，全部搞定了。

不光搬过来了，AI还顺手修了两个他当年自己都没发现的bug。更离谱的是，他27年前有一个特别想做但做不出来的可视化工具，一个「Minkowski空间的Inkscape」。当年代码写到一半发现复杂度太高，放弃了。结果这次他跟AI聊了几个小时，做出来了。

我看完这篇的第一反应不是「AI好厉害」，而是另一个想法。这背后可能藏着一个被绝大多数人忽略的副业方向。

「App考古」。

你想想看，Terry Tao碰到的这个问题，根本就不是他一个人的。1999年到2010年之间，全世界有多少大学老师、研究人员、机构，用Java Applets、Flash、Silverlight做了大量交互式教学工具和可视化程序？这些东西现在几乎全废了。但那些内容的价值还在。老师们还在用同样的课件讲同样的知识点，学生们还在学同样的概念。只是那些本来能让学习体验好10倍的可视化工具，全都没法用了。

我顺着这个思路往下捋了捋，发现至少有三个可以切入的方向。

第一个方向，帮研究人员做论文配套可视化。这个门槛最低但需求最刚。越来越多的论文开始配套交互式可视化。以前只有计算机专业的人能做，现在任何一个博士生都能用AI Coding Agent给自己做论文配套演示。Terry Tao自己写完那篇Gilbreath猜想的论文后，顺手用AI配了一个可视化，现在他发论文都打算标配一个。这个需求在学术圈会越来越强。你帮一个博士生做论文配套可视化，收800到2000很正常。工科、数学、物理、生物信息学都可以做。我之前在行业经验才是护城河那篇聊过，懂行的人用AI才是最值钱的。做论文可视化，关键是你理解论文在讲什么，AI帮你写代码。

第二个方向，帮大学和博物馆迁移老旧交互内容。这个利润更可观。很多大学的教学页面、博物馆的交互展项，底层还是Flash或Java Applets。我在HN评论区看到有人问「AI Coding Agent是不是更擅长维护旧代码」，评论区一阵共鸣。说真的，维护旧代码这种活，程序员不太愿意接，因为不够酷。但AI不在乎酷不酷，你给它旧代码，它就能出迁移方案。一个完整的Flash交互展项迁移到HTML5，收5000到2万都有可能。跟之前聊的Vibe Coding做App不一样，做App考古面对的是有预算的客户，大学、博物馆、研究所，这些人付得起钱，而且不会跟你磨叽。

第三个方向，也是最大的机会，企业内部遗留系统现代化。这个最炸但也最需要耐心。说实话，很多公司内部还有大量基于jQuery、ExtJS甚至Silverlight搭建的管理系统。20年前的代码，没人敢动，也没人有空动。但用AI Coding Agent迁移这些东西，效率和安全性可以兼顾。Terry Tao在文章里提了一句让我印象特别深的，他检查了所有迁移后的代码，只发现了一个细微的拖拽事件bug，而且AI还帮他找到了原始代码里两个他自己没发现的bug。AI迁移后的代码质量，至少不比他24年前手写的差。如果AI连一个菲尔兹奖得主24年前的Java Applets都能无损迁移，企业内部遗留系统的迁移就更没问题了。这个方向的单子从3万起步，但你需要懂一点企业的业务流程。如果你之前在企业做过IT或者对接过业务系统，这就是你的红利窗口。

三个方向一层一层往上走。论文可视化最轻量，随时可以开始。大学博物馆迁移中等，利润和竞争比都很好。企业遗留系统是终局，天花板最高。

需要的工具也不复杂。Claude Code或者OpenCode都行，再加一个基础的HTML/CSS/JS理解。我记得之前有人问我，不会JS能不能做，我跟你说，你都不需要自己会写框架级的代码，AI帮你干了90%，你只需要做质量验收和调试就好。。。

大时代啊朋友们。一个数学家27年没做出来的东西，AI几个小时做完了。那些躺在21世纪初互联网废墟里，还等着被人捡起来的App，比我们想象的要多得多。磨平一些信息差。

❓ 常见问题

不会编程的人能做AI App考古副业吗？

做迁移工作需要一点基本的代码理解和调试能力，但不需要你是全栈工程师。核心能力其实是两样，一是能看懂旧代码想干什么（这个AI能帮你分析），二是能验收AI生成的代码质量对不对。你不需要自己写框架级别的代码，AI会帮你完成大部分工作。建议从一个你熟悉的领域的旧项目开始练手。

App考古副业怎么找到第一个客户？

最直接的方式是去大学网站上找到那些打不开的教学交互页面，主动联系相关课程的教授。跟他们说，您看这个Applet已经失效了，我可以帮您迁移到现代浏览器。第一单可以不收钱或者收个友情价，有了案例展示之后，后续的客户会自己找上门。博物馆和研究所同理，这些单位有预算但找不到人做这件事。

Terry Tao用的什么AI Coding Tool？

他在博客里没有明确说具体用的哪一款agent，但从描述来看应该是Claude Code。目前市面上主流的AI Coding Agent就是Claude Code、OpenCode和Cursor。这三款的能力都足够做代码迁移工作，Claude Code在理解复杂代码逻辑上更稳一点。工具不是瓶颈，你用它去做才是。
