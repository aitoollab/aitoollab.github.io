用autopreso做演讲交付：边说边画的白板AI，187星的新玩法

2026-05-11 · AiToollab
autopreso 语音演讲 白板PPT 副业
你有没有过这种感觉，准备一场分享，光是搭PPT就耗掉了大半精力？内容其实都在脑子里，但要把它们变成「能演示的东西」，那个转化的过程又累又费时间。我最近在GitHub上刷到一个小工具，叫 autopreso，解决的就是这个问题——你说，它画，实时生成白板演示。真的，我第一眼看到slogan的时候就觉得有意思：Let the whiteboard whiteboard itself. 让白板自己画自己。5天时间，187颗星，增长速度挺猛的。

点进去看了一下README，我就知道这东西为什么能冲上Trending了。

它解决的是什么问题？

传统的演讲准备流是这样的：先写内容，再做PPT，最后排练。整个流程是线性的，内容生产和视觉呈现是分开的两个环节。 autopreso把这两个环节合并了。你对着麦克风说话，它实时把你说的内容转成白板上的图形、箭头、文字标签。相当于你一边讲，一边有人在帮你画思维导图，而且是那种手绘风格的白板图。 工作原理是这样的：

核心技术栈
麦克风 → 语音转文字(STT) → AI Agent → Excalidraw白板
全程本地运行，语音数据不传服务器。支持OpenAI、Codex、Ollama作为AI大脑。

它底层调的是Excalidraw的API，你说什么内容，它就让白板上对应地长出什么形状来。讲用户增长，它就画一条上升曲线；讲三个策略，它就给你排成三列加箭头。有点像是给演讲者配备了一个「会动的思维导图生成器」。

安装很简单，一条命令：

npx autopreso

没有npx也行，npm install -g autopreso装成全局命令也行。跑起来之后自动打开浏览器，界面就是一个Excalidraw白板。你在左边选麦克风、选AI模型，右边就是白板区域。点「Start Preso」，开讲，白板就开始自己长内容了。

为什么它值得关注？

这个东西有意思的地方在于，它解决的不是「效率问题」，而是「心态问题」。很多人怕演讲，不是因为不知道讲什么，而是因为一想到「要做PPT」就累了。autopreso把PPT这个环节去掉了，你只管讲，白板帮你视觉化。 对于以下几类人，这个工具的吸引力是真实的：

第一类是知识博主和在线课程讲师。他们需要持续产出内容，但PPT制作本身就占掉了大量时间。用autopreso，录一期分享的前期准备可以压缩到「想清楚要讲什么，然后对着麦克风讲」的程度。

第二类是培训师和企业内训师。他们讲课的特点是需要互动、需要画图，但大部分时间是在讲而不是在画。autopreso让边讲边画变成了零成本的操作。

第三类是有分享习惯的创业者。一个CEO说要每周做一次内部分享，如果每次都要花3小时做PPT，可能两周就放弃了。但如果5分钟能搭一个白板框架，然后边讲边生成，这个习惯就更容易维持。

怎么用它赚钱？

这是我重点想的部分。一个工具能不能变现，看的不是工具本身，而是它解决的那个问题背后有没有人愿意付钱。autopreso解决的是「演讲交付效率」的问题，愿意为高效演讲付钱的人，付费能力和意愿都不差。 我有5个方向的思路，说给你听。

方式一：卖「种子素材包」

autopreso支持在演讲前先在白板上放一些「种子元素」，作为AI绘图的基础。你可以在这个基础上，针对不同行业、不同场景，预制不同的种子包。

比如一个「商业计划书演讲包」，里面包含了一个基础框架——封面布局、三段式结构、融资节点标注框等。一个「产品发布会包」，包含产品功能分类、对比表格框架、用户痛点标注。一个「管理培训包」，包含组织架构图、流程图模板、考核指标框。

预估收入：99-299元/套，面向B端可以卖企业授权，999元/年起。

方式二：帮人定制「Agent指令集」

autopreso白板上画什么，取决于AI Agent的指令。你可以把「讲什么内容AI会画什么图形」理解为一个Prompt工程。

不同的人对着同一个autopreso讲，AI画出来的东西质量可以差很远。有些人讲了一大段，AI画出来的是一坨混乱的形状；有些人简单几句话，AI画出来的结构清晰、重点分明。这中间的差别就是「Agent指令调优」的价值。

你可以提供定制服务，帮客户优化他们的autopreso Agent指令，让AI更懂他们的演讲风格和内容逻辑。按次收费，每次500-2000元不等。

预估收入：月接10-15单，月入5000-20000元。

方式三：企业培训「演讲效率」服务

企业内部培训、会议演讲、汇报这些场景，用autopreso可以大幅降低演讲准备成本。你可以做企业培训，教他们怎么用autopreso来做内部知识分享。

这个方向的收费模式可以是一天的企业培训Workshop，报价5000-15000元。内容是：autopreso工具介绍、演讲结构设计、种子素材制作、实操演练。

预估收入：每月2-3场企业培训，月入10000-30000元。

方式四：知识付费内容演示

如果你自己在做知识付费产品，可以用autopreso来做演示。现在很多人买课之前会先看免费分享，你在B站、小红书、视频号发一段「用autopreso讲一个知识点」的视频，观众能直观看到这个工具的价值。

autopreso生成的演讲效果本身就带有「展示性质」，很适合短视频传播。你不需要露脸，只需要展示白板上的内容「从零到有」地生长出来，这个过程就足够吸引人了。

预估收入：流量变现+引流到付费课程，看个人IP影响力，月入不确定，但边际成本极低。

方式五：集成工作流服务

autopreso生成的演讲是Excalidraw格式的白板，可以导出成图片、SVG，甚至PDF。如果你能把它集成到更完整的工作流里——比如演讲后自动生成带你配音的图文版、自动发布到Notion或者飞书文档——这个集成服务的价值就更高了。

做一个「autopreso + 自动化发布」的服务包，帮客户实现「讲完自动出全套物料」。报价3000-8000元/套。

预估收入：每套5000元左右，月出2-3单，月入10000-20000元。

我的判断

坦率地讲，autopreso现在还是alpha版本，功能上还有很多粗糙的地方。但正因为是早期，才有时间窗口。等它成熟了、社区大了，机会反而可能就少了。

这个工具本质上是把「演讲」和「可视化」合二为一，解决了内容创作者、培训师、知识工作者最痛苦的那个环节——从脑子里的东西到能展示的东西之间的转化成本。只要这个问题存在，解决方案就有价值。

现在入场最好的姿势可能不是等它完美，而是用起来、分享出去、在它长大的过程中找到自己的位置。工具不完美的时候，也是做内容、做服务最好做的时候。等所有人都觉得这东西牛的时候，就轮到卷价格了。
