说说这个团队是什么来头。nexu-io/open-design，40K星，200多个贡献者，是目前最大的AI设计系统开源项目之一。html-anything就是他们在这个方向上做的一个垂类工具。所以技术底子是完全过硬的，不是那种跑一圈就没了的东西。

它具体能做什么？

75个模板，9种输出类型。我挑几个你们可能用得上的：

演讲PPT——瑞士国际风格、贵气编辑风、XHS马卡龙色系，一共20种 keynote 风格可选。你给AI一段文字，它直接给你生成一套可以演示的幻灯片。

小红书卡片——专门优化过的配图格式，一键导出2倍分辨率PNG，直接粘贴到小红书编辑器里。

微信公众号长文——杂志风格的排版，支持多种配色方案，写完直接复制粘贴。

个人简历——现代简约风格，AI根据你的信息自动排版，输出可直接打印的PDF友好格式。

还有个很骚的——视频封面脚本。直接输出 Hyperframes 格式，可以丢给 HeyGen 或者 Remotion 渲染成视频。

你说这些功能微喀市场上有没有？都有。但问题是，这玩意零门槛。你不需要学任何设计，不需要装任何插件，不需要付任何订阅费。你只要电脑上有个 Claude Code 或者 Cursor（这两个都有免费额度），就能跑。

怎么用它赚钱？5种方式

方式一：内容创作服务（最容易启动）

你去闲鱼或者小红书发个广告，就说「代做高颜值PPT/演讲稿/图文卡片」。报价参考：普通PPT 80-150一页，复杂演示文稿 300-800一套。你用html-anything生成，速度比传统方式快3-5倍。客户给文字，你给成品，中间全是AI干的。

我认识一个做小红书的朋友，她帮人代运营图文账号，一篇笔记报价300-500。用html-anything制作配图，单张成本基本为零，每天可以接3-5单。这就是纯利。

方式二：垂直行业模板包

html-anything是开源的，你可以fork它的模板，改成你自己的版本。比如，专门做「律师用的法律服务文书模板包」，或者「健身教练用的训练计划可视化模板」。

把这些模板打包，放到Gumroad或者自己的网站上卖。参考价：19-49美元一套。这个项目的75个模板是通用的，你只需要针对某个垂直场景做深度定制，就能建立差异化。

方式三：企业宣传物料服务

很多中小企业需要做产品介绍、公司介绍、招商手册，但请不起设计公司。一份20页的招商手册，设计公司报价通常在5000-20000。而你用html-anything + AI内容创作，2-3天就能交付，价格可以压到1500-3000。

核心逻辑是：AI承担了排版和视觉设计的工作，你负责和客户沟通需求和组织内容。这里面的手艺在于「prompt写得准不准」，跟设计能力关系不大。

方式四：培训课程

教你目标用户「用AI编程工具做内容创作」。対象是：内容创作者、小红书博主、个体户老板。课程内容可以是：

第二课，用Claude Code生成第一张图文卡片（30分钟）
第三课，一键导出小红书可用配图（20分钟）

录播课定价99-199，直播课可以卖到499-999。这个项目的独特性在于「来自40K星团队的技术背书」，跟其他野路子AI课比起来，用户信任度完全不一样。

方式五：做模板订阅服务

每月更新一批新模板，用户付月费订阅。比如「每月69元，获得20套新模板 + 社群答疑」。这个模式的精髓是「懒人包」——用户不需要研究，直接用。

你可以先在GitHub上免费发一套基础模板包引流，然后付费版提供进阶模板和数据更新。你看 html-anything 本身是开源的，但「帮你整理好、测试好、加上使用说明」这件事是有价值的。

我的判断

先说这个项目本身的吸引力。

最大的差异化在于：它是目前唯一一个把「AI编程工具」和「内容创作」打通的工具。Claude Code、Cursor这些工具，用户量非常大，但普通人用不上——因为它们本来就是给程序员写的。这个项目降维打击，让会写prompt的人也能用上这些工具。

而且它的商业路径很清晰——不收费工具本身，靠周边服务赚钱。这和它在GitHub上的爆发是匹配的，因为开发者喜欢这种模式。

但也有坑我想提醒你们。

先说学习曲线。这玩意它宣称零门槛，没骗人——装上确实能跑。但想跑出客户想要的东西，得花时间摸索模板和prompt。我的建议是，先自己玩两周，确认这东西真的顺手了，再考虑拿出去卖钱。别急着发广告，砸自己招牌。

再说市场竞争。内容创作这个市场是真的卷。你要面对的不仅是其他用html-anything的人，还有Canva、Figma、各种AI PPT工具。差异化在哪儿？「垂直场景」和「服务质量」，而不是「会用这个工具」。这么说吧，你得找到自己的细分领域，而不是泛泛接单。

最后说付款意愿。AI内容创作的需求是真的有，但客户分层很明显。有些人愿意为一份精美的招商手册付几千元，有些人觉得AI生成的就是不值钱。前者才是你的目标客户，后者你花再多时间也转化不了。

怎么说呢，我觉得这个项目的窗口期大概还有3-6个月。现在入局做服务，时间上还来得及。但如果是卖课或者模板包，需要尽快建立品牌认知，否则等大厂跟进就没有机会了。
