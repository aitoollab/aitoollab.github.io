AI编程入门教程 零基础用Claude Code和ChatGPT做网页小工具

2026-07-13 约1050字 阅读7分钟
AI编程入门 零基础编程 Claude Code AI写代码

说实话，我以前也觉得编程门槛很高。什么HTML CSS JavaScript，看教程看到第三行就犯困。后来发现AI已经把门槛踩平了。你不用懂语法，不用背API。你只需要清楚自己想要什么，然后让Claude Code或ChatGPT帮你写出来。我最近用Claude Code做了好几个小工具，从没碰过代码的朋友照着我的方法也做出来了。今天把流程完整拆开。你照着走一遍，保证能做出来自己的网页。

第一步，想清楚你要做什么

这一步最容易被忽略。很多人上来就让AI写个网页，结果AI出了个企业级的什么系统根本用不上。正确做法是想一个特别小特别具体的需求。比如做一个记账的小页面。或者一个待办事项清单。或者一个帮你算BMI的身体指数计算器。越具体越好。我自己的第一个AI网页项目，就是一个每日喝水记录的页面。点一下加一杯水，显示今天喝了多少。功能简单到不行，但做出来特别有成就感。

注意，不要一上来就想做复杂的。先做一个单页的、不需要数据库的、纯前端的东西。因为这样的东西AI一次就能写完，你打开浏览器就能看到效果。积极性一下就来了。如果上来就想做个电商网站，AI写出来的东西你根本不知道怎么跑。做出来没看到效果的话，很容易就放弃了。我见过太多人死在这一步了。。。

确定好做什么之后，把需求写下来。一两句话就行。比如做一个每天的任务清单，可以添加和勾选已完成的任务。简单吧。就这么一两句话，丢给Claude Code或ChatGPT，它就能写出一个完整的HTML文件。

第二步，让AI把网页写出来

这就是最神奇的一步。打开Claude Code或ChatGPT，把你的需求告诉它。你也可以加上一些要求。比如风格要简洁好看一点，手机也能打开，深色模式等等。我看过很多人第一次看到AI输出一整段HTML代码时的表情。我自己当时也差不多的表情。就一行需求，出来了一个完整的、能用的、还带样式的页面。双击文件，浏览器直接打开。能用了。

要是不满意？直接说。颜色不好看就让它换。排版不对就让它改。按钮太小让它调大。这就是AI编程最大的优势。你不用看代码就能改东西。你想想，以前一个不懂编程的人想改网页，得学HTML学CSS。现在直接说就行。我经常改四五版才满意，每次都是一个自然语言指令的事。

有个小坑。AI第一次写出来的代码有时候会有些小问题。比如图片显示不出来，或者某个功能点了没反应。不用慌。直接把问题描述给AI，说这个点了没反应帮我修修。它自己debug自己修。我遇到最离谱的一次，AI写了一个计算器，加减乘除都能算，但是不能连算。我说你帮我修一下，它看了两秒给了我新的版本，修复了。这就跟你雇了一个随时待命的程序员一样，还不用工资。。

第三步，发布出去让更多人用

页面做好了怎么让别人也能看到？其实很简单。不需要买服务器也不需要懂部署。最简单的办法是注册一个GitHub账号，把文件上传到GitHub Pages。打开设置，开启GitHub Pages功能。等两分钟，你的网页就有一个公开的网址了。把链接发给朋友。真的能打开。

你就想想，整个过程下来。你没有写一行代码，没有学任何语法，没有买任何服务器。就靠跟AI聊天，做出了一个可以用的网站。说实话，放到五年前你敢信？但要警惕一点，AI生成的代码毕竟是机器写的。涉及用户数据、登录、支付这些场景千万不要直接上线。只是做个工具给自己和身边人用是完全够的。
