# AI副业提示词全家桶

> 涵盖10大副业方向，从方向筛选到变现路径，提供可直接使用的结构化提示词。本地优先、脱敏设计，无需联网即可部署到生产环境。

---

## 📂 使用说明

所有提示词均基于 Markdown 格式，结构统一：
1. **角色定义** — AI扮演的身份
2. **输入变量** — `{}` 包裹，需要你填入自己的信息
3. **输出要求** — 明确说明输出格式和约束

适用工具：ChatGPT / Claude / 文心一言 / 通义千问 / Kimi / 智谱清言

## 一、副业方向选择

### 1. 精准方向筛选器

```
你是一位资深AI副业顾问，专注于帮普通人用AI工具从零开始建立可持续副业收入。

我的基本情况：
- 职业背景：{工程师/设计师/销售/教师/学生/自由职业/其他}
- 核心技能：{具体技能，如：写作/数据分析/编程/外语/无特殊技能}
- 每周可投入时间：{X}小时
- 初始预算：{0-500元/500-2000元/2000元以上}
- 风险偏好：{保守/稳健/激进}
- 目标收入：{月入1000+/月入5000+/月入10000+}

请从以下10个方向中筛选出最适合我的3个，并说明理由：

1. AI提示词产品销售（边际成本为零，一份多卖）
2. 内容创作变现（小红书/知乎/公众号）
3. AI自动化脚本服务（帮企业做流程自动化）
4. 垂直领域AI服务（简历优化/法律咨询/医疗助手等）
5. 知识付费教程（把经验变成电子书/专栏）
6. 本地商家AI赋能（帮实体店做智能客服/营销）
7. MCP服务器开发（AI工具生态的中间件）
8. AI培训与企业内训（企业AI提效需求爆发）
9. 跨境AI服务（外贸/跨境电商文案+运营）
10. Agent定制服务（帮企业搭建Multi-Agent系统）

对每个推荐方向，请给出：
1. 为什么这个方向适合我（结合我的背景）
2. 最快多久能出第一单
3. 需要避开的坑（1-2个）
4. 下一步最小行动（3步以内）
```

### 2. 副业可行性评估

```
你是一位投资人，专门评估副业项目的可行性。

请分析以下副业想法，给出是否值得投入的判断：

副业类型：{类型}
我的切入点：{具体描述}
目标人群：{谁会买}
定价策略：{定价多少}
独特优势：{为什么你能做别人做不了的}
初始投入：{时间和金钱}

评估维度：
1. 市场需求（1-10分）：目标用户是否真的有付费意愿
2. 竞争壁垒（1-10分）：你能做多久，别人多久会复制
3. 变现速度（1-10分）：多久能收到第一笔钱
4. 可规模化（1-10分）：能否从兼职变成稳定收入
5. 总评：投入产出比

最终给出三个结论：
- 立刻做：原因
- 先等等，需要解决X问题
- 不做，换一个方向
```

### 4. 知乎回答引流模板

```
你是一位知乎高赞回答者，擅长用结构化思维回答问题并自然植入软广。

问题：{知乎问题原文}
我的背景：{我的真实经历或专业背景}
产品/服务：{我想要推广的产品}

要求：
1. 先给出专业、有价值的回答（300字以上，干货为主）
2. 在第3-4段自然植入推广信息（不是硬广，是"我用过所以推荐"）
3. 结尾给出行动指引
4. 全程语气客观，不夸张

回答结构：
- 开篇：直接回应问题，给出核心结论
- 主体：2-3个有数据或案例支撑的分论点
- 软植入段：自然过渡到产品经验
- 结尾：开放性问题引导评论

注意：全文不出现"绝对"、"保证"、"第一"等绝对化用语。
```

## 三、垂直领域服务类

### 7. 简历优化诊断报告

```
你是一位资深HR，有着10年简历筛选经验。

请帮我分析以下简历的问题，并给出修改建议。

我的简历：
{粘贴简历全文}

目标岗位：{岗位名称}
目标公司类型：{大厂/中小公司/创业公司/外企}

分析维度：
1. 岗位匹配度：简历与目标岗位的匹配程度（1-10分）
2. 量化程度：数据化表达是否充分（1-10分）
3. 逻辑清晰度：工作经历叙述是否清晰（1-10分）
4. 关键词密度：是否命中简历筛选系统（ATS）（1-10分）
5. 最大问题：当前简历最致命的1-2个问题

针对每个问题，给出：
- 问题描述
- 修改前/修改后对比示例
- 修改理由

最后输出一份优化后的简历摘要（100字以内，用于简历开头的一句话自我介绍）。
```

### 8. 法律咨询初筛助手

```
你是一位民商法咨询顾问，擅长用通俗语言解释复杂法律问题。

用户问题：{用户描述的法律问题}
背景信息：{相关背景，如合同金额、双方关系、时间节点等}

请进行初步法律分析：

1. 问题定性：这个情况属于哪类法律关系（合同/侵权/劳动/婚姻等）
2. 法律依据：涉及哪些法条（用通俗语言解释，不必引用条文编号）
3. 责任判断：在没有诉讼的情况下，预判责任归属
4. 维权路径：下一步可以采取什么行动
5. 风险提示：诉讼/仲裁的时间成本、经济成本、胜诉概率

重要免责声明（必须包含）：
"以下仅为初步分析，不构成正式法律意见。具体法律问题请咨询执业律师。"

适用场景：法律科普账号自动回复、付费咨询前置筛选、律师获客引流。
```

### 9. 竞价页面转化率诊断

```
你是一位转化率优化专家，擅长诊断销售页面的问题。

请分析以下销售页面的转化率问题。

页面类型：{落地页/详情页/课程销售页/产品页}
目标转化：{注册/购买/留资}
当前数据：{访问量/跳出率/转化率（如有）}

页面内容或链接：{页面URL或粘贴页面文字内容}

诊断维度：
1. 标题钩子：是否在3秒内传递核心价值
2. 信任背书：是否有足够的证明（案例/数据/背书）
3. 痛点共鸣：是否精准描述用户最关心的问题
4. 行动引导：CTA是否清晰、足够突出
5. 价值感知：价格与价值的对比是否清晰

对每个维度给出：
- 问题（用具体位置/具体语句说明）
- 原因（为什么这里影响转化）
- 修改建议

最后给出一个优化优先级排序（最重要的1-3个改动）。
```


## 六、数据与调研类

### 16. 竞品监控报告

```
你是一位市场分析师，擅长竞品分析。

请帮我分析以下竞品：

竞品名称：{竞品名称}
竞品链接/介绍：{链接或描述}
我的产品：{我的产品定位}

分析维度：

1. 核心卖点：竞品最核心的价值主张是什么
2. 定价策略：竞品的定价结构和逻辑
3. 目标用户：竞品主要服务哪类用户
4. 获客渠道：竞品主要通过什么方式获取用户
5. 用户评价：用户对竞品最满意和最不满的是什么（从App Store/小红书/知乎等抓取）
6. 差异化机会：竞品做得不好但用户又在抱怨的地方

给出一个总结：
- 我们应该借鉴竞品的3个优点
- 我们应该避免竞品的3个坑
- 我们的差异化定位建议

报告格式：每个维度300字以内，最后给出行动建议。
```

## 附录：提示词使用技巧

### 如何让AI更懂你

1. **角色先行**：先告诉AI它是谁，再描述任务
2. **变量要具体**：{}里的内容越具体，输出越精准
3. **给出示例**：告诉AI"像这样"比"写得更好"更有效
4. **限制输出**：加"不超过200字"比不限制效果好3倍
5. **迭代优化**：第一次输出后，说"更有深度"或"更口语化"让它调整

### 常见错误

- ❌ "帮我写一篇好文章" → 结果泛泛
- ✅ "帮我写一篇针对25-30岁程序员的小红书帖子，关于用AI做副业，800字，语气轻松有梗"

- ❌ "给我一些创业建议" → 结果放之四海皆准
- ✅ "我是一个初中数学老师，想在课余做副业，每周8小时，预算500以内，请给我3个最可行的方案"
